# claude-usage-peek

[English](README.md) · [中文](README.zh.md) · **日本語**

> [Claude Code](https://claude.com/claude-code) だけで開発しました。

macOS のメニューバーから Claude の使用量をサッと確認できるツールです。

- **あなたの情報を一切取得しません** —— トラッキングも分析もなし。あなたに関する情報が Mac の外へ出ることはありません。
- **軽量** —— 小さなメニューバーアイコンだけ。`pip install` 不要、バックグラウンドで重い処理もしません。
- **安全** —— 読み取り専用。Claude のデータを変更することは一切ありません。

メニューバーに 🤖 アイコンが常駐し、5 時間枠の残り利用量を表示します。クリックすると
5h / 7d の利用上限とリセット時刻を出す小さなパネルが開き、さらにグラフやヒートマップ付きの
HTML ダッシュボードに展開できます。

> **🔒 ローカル・安全。** すべての集計はあなたのマシン上で完結します。**使用量データは
> アップロード・外部送信・第三者経由を一切しません。** Claude のデータも変更しません。
> 唯一のネットワークアクセスは、あなた自身のログイントークンで**公式 `api.anthropic.com`** に
> 一度だけ実際の利用上限% を**読み取る**ことだけです（読み取り専用・データ送信なし・
> トークンはディスクに保存も出力もされません）。

## 必要環境

- **macOS 13+**
- **Python 3** —— 使用量と利用上限を読み取るエンジン（PATH に `python3` があること）
- **Xcode コマンドラインツール**（app を一度ビルドする `swiftc` 用）——
  `xcode-select --install` でインストール

> `pip install` 不要、第三者パッケージなし、アカウント不要、設定不要。

## インストール

1. **このフォルダをダウンロード**（clone または ZIP）。
2. **app をビルド** —— Finder で `build_menubar.command` をダブルクリック、または
   ターミナルで：
   ```bash
   bash build_menubar.command
   ```
   **表示言語**（English / 中文 / 日本語、デフォルトは英語。後から変更可）を尋ねられ、
   ビルド後にデスクトップへ **Claude Usage Bar.app** が作られます。
3. **app を開く** —— `Claude Usage Bar.app` をダブルクリック。メニューバーに 🤖 が出ます。

> フォルダを移動した場合は、`build_menubar.command` をもう一度実行してください。
>
> ログイン時に自動起動：**システム設定 → 一般 → ログイン項目** に app を追加。

## 使い方

- 🤖 を**左クリック** → パネルが開きます：
  - **5時間ウィンドウ** と **7日間ウィンドウ** の進捗バー（塗り＝使用量、緑→橙→赤）、
    **使用%**、**リセット予定時刻**、そして**この調子だとリセット時に何%になるかの予測**
  - **🔄 更新** —— 公式の利用上限を取り直す
  - **📊 ダッシュボードを開く →** —— ブラウザで完全な HTML ダッシュボードを開く
- 🤖 を**右クリック** → 小さなメニュー：**利用上限を更新** / **ダッシュボードを開く** /
  **言語（Language）** / **終了**。

### 言語の変更

インストール時に言語を選びます（デフォルトは英語）。後から変えるには、**🤖 アイコンを
右クリック → 言語（Language）** で **English / 中文 / 日本語** を選択。すぐに切り替わり、
次回起動時も記憶されます。再ビルドは不要です。

## 数値の意味

- **5h / 7d %** = **Anthropic アカウント全体**の公式上限（Web・デスクトップ・Claude Code・
  API —— Anthropic 共通の上限）を公式 API から読み取った値。
- **ダッシュボードのグラフ** = **このマシンの *Claude Code* 使用量**（ターミナル CLI、
  VS Code やその他 IDE 拡張 —— `~/.claude/projects/` にセッションを書き込むもの全般）。
  claude.ai の Web 版や Claude デスクトップチャット app は**含みません**。

ダッシュボードは自己完結型の HTML ページ（JavaScript なし・CDN なし）：今日 / 今週 /
今月 / 累計トークンのカード、リセットカウントダウン付きの公式 5h/7d バー、GitHub 風の
日次ヒートマップ、時間別の棒グラフ、モデル別の内訳。ローカルサーバーは `127.0.0.1` のみで待ち受けます。

## アンインストール

```bash
bash uninstall.command   # または Finder でダブルクリック
```

メニューバー app とバックグラウンドサービスを終了し、デスクトップの app・キャッシュ・
ログ・言語設定を削除し、（任意で）フォルダも削除します。システムレベルのインストールは
一切していないので、削除すればきれいに消えます。

## 上級者向け —— コマンドラインツール（任意）

app のエンジンは、直接実行もできる監査可能な小さな Python スクリプトです：

```bash
python3 usage.py            # 日別＋モデル別のトークン集計（--json で生データ）
python3 dashboard.py        # HTML ダッシュボードを生成して開く
python3 dashboard.py --serve  # ライブダッシュボード http://127.0.0.1:8787
python3 quota.py            # 公式 5h/7d% を表示（api.anthropic.com へ 1 回）
python3 watch.py            # バックグラウンド監視：50/75/90% で macOS 通知
```

`usage.py` を Claude Code ターミナルの **statusline** に組み込むこともできます ——
[usage.py](usage.py) を参照。（statusline はターミナル CLI のみで表示され、VS Code パネルには出ません。）

## セキュリティ

- 使用量の集計はローカルファイルの読み取りのみで**オンライン通信なし**。唯一通信するのは
  `quota.py`：このマシンの Claude Code ログイントークンで **api.anthropic.com**（公式）に
  1 回だけリクエストし、実際の利用上限% を読みます。**第三者を経由せず、トークンはディスクに
  保存も出力もしません。** `~/.claude/projects` 下のデータは変更しません。
- メニューバー app は利用上限キャッシュ（`~/.claude/usage-peek-quota.json`）の読み取りと
  同梱 Python スクリプトの起動のみを行い、それ自体はネットワーク通信をしません。
- 第三者依存なし、監査可能な単一ファイル：[usage.py](usage.py)、[quota.py](quota.py)。
